  • Герои и армии — собирайте отряды из различных существ, захватывайте замки и ресурсные объекты
  • Транспортная система — используйте корабли и летающие транспорты для перевозки войск через непроходимую местность

  • Обмен сохранениями — переносите сохранения между браузерами и устройствами

Многопользовательский режим через обмен сохранениями

Игра поддерживает многопользовательский режим в стиле ранних стратегий — через обмен файлами сохранений. Каждый игрок делает свой ход, сохраняет игру и отправляет файл следующему игроку. Идеально подходит для игры с друзьями через почту или мессенджеры!

Управление:

  • ЛКМ — выбор и перемещение юнитов / перетаскивание для панорамирования карты
  • Колесо мыши — масштабирование
  • Пробел — завершить ход
  • TAB — следующий герой

Нужно ли устанавливать игру? Нет, все игры работают прямо в браузере без установки.

Нужна ли регистрация? Нет, игры доступны без создания аккаунта.
